# PDFs de Entrada

Coloque os arquivos PDF de editais neste diretório para processamento.

## Uso

```python
from main import EditalProcessor

processor = EditalProcessor()
processor.process('input_pdfs/edital.pdf')
```

## Estrutura sugerida

```
input_pdfs/
├── edital_2024_001.pdf
├── edital_2024_002.pdf
└── README.md
```
